Cos'è Firefox OS

Firefox OS (Boot to Gecko) è un sistema operativo basato su kernel Linux, utilizza standard completamente aperti e il suo codice è completamente open source, anche nelle fasi di sviluppo delle varie versioni

https://developer.mozilla.org/it/Firefox_OS

Gonk, Gecko e Gaia

Firefox OS si sviluppa su tre differenti livelli:

  1. Gonk - Il livello più basso, Kernel Linux (basato su AOSP) e driver open source
  2. Gecko - Il motore grafico e la Javascript Virtual Machine
  3. Gaia - L'interfaccia utente, completamente realizzata in HTML, CSS e Javascript

https://developer.mozilla.org/en-US/docs/Mozilla/Firefox_OS/Platform/Architecture

Hosted vs Packaged

Firefox OS supporta due diverse tipologie di Web App: Hosted e Packaged, con alcune differenze...

  • Privilegi differenti (Geolocalizzazione, Storage, Sveglia, Radio, Notifiche)
  • Funzionalità differenti (Lettura/scrittura SD, Online/Offline)
  • Modalità di aggiornamento differenti
  • ...

Permessi

Su altri sistemi siamo abituati ai permessi di amministrazione. Firefox OS specifica i permessi di ogni app nel suo manifest e se permesso dall'utente verranno eseguiti.

Con Firefox OS i permessi di root servono solo agli sviluppatori per il flash delle ROM e il debug delle applicazioni di sistema e sono attivabili solo in presenza di un kernel compilato con l'opzione abilitata

https://developer.mozilla.org/en-US/docs/Mozilla/Firefox_OS/Security

Manifest

Il manifest.webapp è un file JSON contenente le informazioni sull'applicazione

  • Nome, descrizione, dati sullo sviluppatore e lingue supportate
  • Percorso di lancio (launch_path) dell'applicazione e le icone
  • Orientamento dell'app e supporto al full-screen
  • webActivity
  • Permessi richiesti

      • https://developer.mozilla.org/en-US/docs/Web/Apps/Manifest

Boilerplate App

Firefox OS Boilerplate App è uno strumento che permette di provare le principali API di Firefox OS e di capirne il funzionamento.

https://github.com/robnyman/Firefox-OS-Boilerplate-App

WebIDE

WebIDE è il sostituto di App Manager. Come App Manager, permette il debug delle applicazioni per Firefox OS usando runtime come i Firefox OS Simulator oppure un dispositivo fisico.

Permette inoltre di connetere i Firefox Developer Tools ad altri browser, come Firefox per Android, Chrome per Android e Safari su iOS.

https://developer.mozilla.org/it/docs/Tools/WebIDE

Sicurezza

Firefox OS è stato sviluppato con l'idea alla base di preservare la sicurezza dell'utente e dei suoi dati.

  • I permessi di qualsiasi app (di sistema e non) sono in chiaro e gestibili completamente dall'utente.
  • Le applicazioni girano sotto sandbox e possono quindi colloquiare con il sistema solo attraverso le API fornite... Niente accesso diretto alla SD!

https://developer.mozilla.org/en-US/docs/Mozilla/Firefox_OS/Security
